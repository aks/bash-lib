Set Display Attributes

Set Attribute Mode      <ESC>[{attr1};...;{attrn}m
Sets multiple display attribute settings. The following lists standard attributes:

0       Reset all attributes
1       Bright
2       Dim
4       Underscore
5       Blink
7       Reverse
8       Hidden
